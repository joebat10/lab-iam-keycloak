# Glossaire rapide

| Terme | Définition courte |
|---|---|
| AuthN | Authentification — vérifier une identité |
| AuthZ | Autorisation — vérifier un droit d'accès |
| IdP | Identity Provider — authentifie et émet des tokens/assertions |
| SP / RP | Service Provider (SAML) / Relying Party (OIDC) — l'application cliente |
| SSO | Single Sign-On — une authentification, plusieurs applications |
| Realm | Domaine de confiance isolé dans Keycloak (utilisateurs, clients, rôles) |
| Client | Une application enregistrée dans un realm Keycloak |
| JWT | JSON Web Token — jeton signé auto-porteur |
| PKCE | Proof Key for Code Exchange — protection du flux Authorization Code |
| Scope | Portée d'un accès demandé/accordé |
| Claim | Une information portée par un token (ex. `sub`, `email`) |
| RBAC | Role-Based Access Control |
| ABAC | Attribute-Based Access Control |
| ReBAC | Relationship-Based Access Control (Zanzibar) |
| IGA | Identity Governance & Administration |
| JML | Joiner-Mover-Leaver — cycle de vie d'une identité |
| SCIM | Standard de provisioning/déprovisioning de comptes |
| DN | Distinguished Name — identifiant unique d'une entrée LDAP |
| Tuple (OpenFGA) | Une relation stockée : `objet#relation@utilisateur` |
